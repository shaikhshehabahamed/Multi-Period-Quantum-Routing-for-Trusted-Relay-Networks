# Multi-Period-Quantum-Routing-for-Trusted-Relay-Networks

Trusted-relay Quantum Key Distribution (QKD) networks face a distinctive routing challenge in which secret keys behave as time-coupled, perishable inventories. Decisions made in the current period directly affect key availability in future periods, creating a dynamic multi-objective problem that must simultaneously reduce unmet demand, latency, and key consumption. Existing classical optimization approaches often yield limited diversity in Pareto-efficient operating points under such time-varying inventory constraints, which restricts an operator’s ability to select solutions aligned with changing priorities. To address this, we propose a hypervolume-guided Quantum Pareto QAOA integrated within a Model Predictive Control (MPC) framework. We formulate multi-period trusted-relay routing as a tri-objective discrete optimization problem and introduce a preference-controlled quantum circuit with XY-mixers that preserve feasibility structure during the search. Across periods, the proposed method can produce multiple non-dominated trade-offs. For example, QAOA-MPC executes (112.775, 5.698, 150.937) in period 3. Overall, the results indicate that combining hypervolume-guided quantum optimization with MPC provides a robust closed-loop approach for dynamic QKD routing, enabling effective management of the trade-offs between service quality and resource exposure.
